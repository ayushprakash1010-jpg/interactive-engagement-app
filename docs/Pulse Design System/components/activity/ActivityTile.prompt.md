Activity-type picker tile (builder). Pass a Lucide icon; `type` color-codes the chip.

```jsx
<ActivityTile type="poll" icon={<BarChart3 size={20} />} title="Live poll" description="Single, multiple, rating, open text" />
<ActivityTile type="ai" icon={<Sparkles size={20} />} title="Generate with AI" description="Describe your session, get a draft agenda" />
```
